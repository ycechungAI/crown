Detailed Description

An abstraction to access data files.

It represents a flow of data attached to a 'file' which can be an archived file, a regular file, a location in memory or anything that can be read or wrote. A File is an abstraction to interact with these in an uniform way; every file comes with a convenient set of methods to facilitate reading from it, writing to it and so on.

virtual void crown::File::flush ( )
pure virtual

Forces the previouses write operations to complete.

Generally, when a File is attached to a file, write operations are not performed instantly, the output data may be stored to a temporary buffer before making its way to the file. This method forces all the pending output operations to be written to the file.

Implemented in crown::NullFile, crown::DiskFile, and crown::ApkFile.

virtual bool crown::File::is_valid ( )
pure virtual

Returns whether the file is valid.

A file is valid when the buffer where it operates exists. (i.e. a file descriptor is attached to the file, a memory area is attached to the file etc.)
