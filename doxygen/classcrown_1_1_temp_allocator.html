Detailed Description

template<uint32_t SIZE>
class crown::TempAllocator< SIZE >

Allocates from a fixed-size buffer.

When the internal memory is exhausted or the allocation size exceeds the remainig internal storage, the backing allocator is used instead. The memory is automatically freed when the allocator is destroyed.
